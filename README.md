# Vic3

This repository contains the original game file of Victoria 3.

Large binary files are ignored, see `.gitignore` for details.
The checksum of those ignored files are listed in `ignored.checksum`, generated by
```bash
find -type f -printf '%P\n' | \
	git check-ignore --stdin | \
	xxhsum -H2 --binary --filelist - > ignored.checksum
```
